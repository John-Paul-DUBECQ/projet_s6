Pour pouvoir lancer le projet, vous devez tout d'abord installer NodeJS en téléchargeant le programme à partir de ce lien : 
https://nodejs.org/en/download
ou
https://nodejs.org/en


Ensuite, il suffit de se rendre dans le dossier à la racine du projet (celui où vous vous trouvez) et d'exécuter dans un terminal :

npm i

Ceci va installer toutes les dépendances dont ReactJS

Une fois l'installation terminée, lancez le serveur local avec la commande :

npm start

 Pour pouvoir accéder au site, il suffit d'aller à l'adresse suivante:

http://localhost:8080/
ou
http://localhost:8080/index
